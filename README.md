从《音游开字母》衍生过来的《开字符》，数据都在data文件夹下，可自定义或组合数据。目前的数据有Arcaea的曲名、曲师名、画师名、搭档名、包名、谱师名，还有数学公式，还有瞎几把搞的几个。

直接跑.py或用pyinstaller生成.exe在dist文件夹内：

```bash
pip install pyinstaller
pyinstaller -F "Letter Opener2.0.py"
```

# 初代版本：

开字符直接输入想开的字符，显示答案用星号"\*"，显示7号用\*7，显示15号用\*\*15，显示114514号用\*\*\*\*\*\*114514（6个星号，虽然正常使用顶多用上两个星号）。
*-字段可以将后续的命令视作取消，\*-a表示关闭打开的"a"，，\*-\*\*14表示将打开的14号关闭。若字符或序号未打开将会报错闪退。慎用、避免去用。

# 2.0版本：

重构并增加了指令实现操作（类似于minecraft的指令），为了减少难度（特别是一些有方正字的题目）添加了技能，技能使用osin打开字符，会判断字符到底是否打开

* 开字符：/ocha abc （开a、b、c，可直接输入abc）
* 关字符：/ccha xyz （关x、y、z，可输入--xyz）
* 显示答案：/oans 479 （打开4、7、9号答案）
* 关闭答案：/cans 136 （关闭1、3、6号答案）
* 逐个打开：/osin (1,2) (3,4) （分别打开第1、3个答案中第2、4位的字符）
* 逐个关闭：/csin (9,8) (7,6) （分别关闭第9、7个答案中第8、6位的字符）
* 设置炸弹模式：/setboom 0 （0关1开，默认关）
* 设置炸弹数：/setb 114514
* 技能空袭：/airp x y （不可多个）打开指定未开字符，周围未开的字符的十字型内概率打开，概率默认1/3
* 技能扫雷：/mine x y （不可多个）在已开的字符周围3×3的格子中，未开的字符概率打开，至少打开一个，概率默认1/3
* 智能施法：/b x y 其实就是判断是否已开来决定用什么技能

字符显示优先级：若答案已开，显示完整答案，否则显示已开字符或打开指定位置的字符
若已有条件导致答案实质上已开，则会将答案设置为开，并让炸弹数+1
